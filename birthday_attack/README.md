# Birthday Attack

## Project
Implement the naïve birthday attack of reduced SM3

## 代码说明
本项目使用Python实现


**文件说明**
- **birthday_attack.py** : 生日攻击代码，根据寻找碰撞的比特长度使用两种思想进行生日攻击
- **SM3.py** ： SM3的python版本简单实现


## 运行指导
birthday_attack.py与SM3.py放于同级目录下，运行birthday_attack.py
